# My Agent Skills

This is my personal collection of agent skills. I've developed these to help me with various methodologies and frameworks that I find interesting or useful in my daily work. 

## Installation

To add these skills to your local environment, run:

```bash
npx skills add https://github.com/almirmcunhajr/skills
```

## Available Skills

### [Google Design Sprint](./skills/google-design-sprint/)
A guide through the 6-phase Google Design Sprint methodology. I built this to help facilitate structured problem-solving sessions, prototyping, and user testing.
