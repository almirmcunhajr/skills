# Google Design Sprint Skill

This skill guides users through the Google Design Sprint methodology, providing a structured framework for solving critical business questions through rapid design, prototyping, and user testing.

## Sources

The methods and methodology in this skill are based on the [Google Design Sprint Kit](https://designsprintkit.withgoogle.com/methodology).

## Installation

To add this skill to your environment, run the following command:

```bash
npx skills add https://github.com/almirmcunhajr/skills --skill google-design-sprint
```

## Overview

The skill covers all 6 phases of the Design Sprint:
1. **Understand**: Align the team and articulate the problem space.
2. **Define**: Synthesize findings and pick a target.
3. **Sketch**: Generate solutions individually.
4. **Decide**: Select the best ideas to prototype.
5. **Prototype**: Build a facade of the experience.
6. **Validate**: Test with real users and gather feedback.

Each phase includes a library of methods sourced from the Google Design Sprint Kit, LUMA Institute, Sudden Compass, and Voltage Control, with a priority on **Core Methods**.
